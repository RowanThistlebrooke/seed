# The vault

Three parts. This folder is one of them.

- **The ledger** lives in Supabase. Every log: a set, a sub, a stock, a franc.
  One table, one shape, so it can be read across everything at once.
- **The equation** lives in `lib/tiles/weights.ts`. Your goals and what each tile
  is worth to them.
- **The context** is this folder. Who you are, the why, the chapter you are in,
  what you decided and why.

Context lives here, in the repo, in git, on purpose. If it only lives on one
laptop, it is not yours.
